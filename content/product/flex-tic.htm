FlexTiC™

The FlexTiC™ IP core is a timer and counter core designed for the Ashet Home Computer™.

It is designed to have a very small footprint to cheaply instantiate it several times in a system.

Ashet Technologies™ is proud to provide you with this new technology and enables you to create consumer and industrial appliances with an ease.

Features

  • Simple 32 bit system interface
  • 64 bit counter with safe 32 bit access
  • Configurable 32 bit prescaler
